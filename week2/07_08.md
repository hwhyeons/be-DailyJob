# be-DailyJob

> 이 회고는 작업량을 보고하기 위한 문서가 아니라,
오늘 내가 어떤 판단을 했고 어떤 개발 행동을 반복하거나 바꿔야 하는지 확인하기 위한 기록입니다.
각 항목은 길게 쓰기보다, 사실/판단/근거/다음 행동이 드러나게 구체적으로 작성하세요.
> 

### 1. 오늘 완료한 것

- 오늘 실제로 끝낸 기능/수정/학습
    - 회원가입 이름 중복 테스트 추가
    - HTTP Response에서 Content-Length 누락 수정
    - 회원가입 과정에서 아이디랑 닉네임 중복 발생 상황에서 HTML 모달 추가

### 2. 가장 막혔던 문제와 해결 과정

- 문제 상황 또는 증상
    - ThreadPoolExecutor의 corePoolSize가 2일 때 사파리 브라우저에서 BrokenPipe 문제
- 처음 추측한 원인
    - BufferedReader의 readLine()에서 대기가 발생하였기에 헤더 파싱이 문제라고 생각함
- 원인을 확인하기 위해 시도한 방법
    - 다른 브라우저에서 테스트
    - 디버그모드로 병목 지점 확인
- 실제 원인
    - 정확한 원인 파악 X
- 해결 방법
    - ThreadPoolExecutor의 corePoolSize를 늘림
